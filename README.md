## Run Locally

First of all, you must clone the project to your own computer.

```bash
  git clone https://github.com/gustavogarciac/dream-structure.git
```

Get into the project's directory.

```bash
  cd dream-structure
```

Install dependencies

```bash
  pnpm install
```

Run the server

```bash
  pnpm run dev
```
